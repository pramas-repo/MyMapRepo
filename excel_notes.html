MODULE 1: EXCEL BASICS

1. Introduction to Data Analysis

What is Data Analysis?

Data Analysis is a systematic process of collecting, cleaning, transforming, and interpreting data to extract meaningful insights, support decision-making, improve business performance, and identify patterns or trends.

Data Analysis Helps In:

  • Improving efficiency
  • Increasing profits
  • Understanding trends
  • Making better decisions
  • Forecasting future outcomes

2. Data Analysis Process (Cycle)

a. Define Objectives

Identify the problem to solve and the specific questions the data must answer.

b. Data Collection

Data sources include Excel sheets, databases, surveys, web data, or manual entry.

c. Data Cleaning / Wrangling

  • Handling missing values
  • Removing duplicates
  • Correcting wrong formats
  • Standardizing data
  • Fixing errors

d. Data Analysis Types

  1. Descriptive: What happened?
  2. Diagnostic: Why did it happen?
  3. Predictive: What will happen?
  4. Prescriptive: What action should we take?

e. Interpretation & Visualization

Create charts, dashboards, and graphs.

f. Communication

Use storytelling and reports (PowerPoint).

3. MS Excel for Data Analysis

Excel Interface Terms

  • Workbook: Excel file
  • Worksheet: Sheet inside the file
  • Cell: Intersection of a row and column
  • Ribbon: Top menu with commands

4. Entering, Editing & Formatting Data

  • Enter data by typing in any cell
  • Edit using F2
  • Format text: Bold, Italic, Underline
  • Align data: Left, Center, Right
  • Auto-fill numeric range using Ctrl + Drag
  • Insert rows & columns
  • Create tables for structured data

5. Sorting & Filtering

Sorting

Sort alphabetically or numerically (A→Z, Z→A).

Filtering

Use Ctrl + Shift + L to toggle filter.

6. Basic Formulas

  • SUM: =SUM(E4:E13)
  • AVERAGE: =AVERAGE(E4:E13)
  • COUNT: =COUNT(E4:E13)

7. Conditional Formatting

  • Highlight values above threshold
  • Color scales
  • Highlight duplicates

8. Data Cleaning

  • TRIM: =TRIM(B2)
  • Remove Duplicates
  • Handle missing values
  • Correct data types

MODULE 2: EXCEL FUNCTIONS

Categories of Functions

  1. Text
  2. Number
  3. Date & Time
  4. Logical
  5. Lookup (Most Important)
  6. Statistical

A. Text Functions

  • UPPER: =UPPER("excel")
  • LOWER: =LOWER("DATA")
  • PROPER: =PROPER("prakash patil")
  • CONCAT: =CONCATENATE(A2," ",B2)
  • LEFT: =LEFT(A1,3)
  • RIGHT: =RIGHT(A1,2)
  • MID: =MID(A1,2,3)
  • LEN: =LEN(A2)
  • TRIM: =TRIM(A2)
  • FIND: =FIND("a",C4)

B. Number Functions

  • SUM: =SUM(C2:C12)
  • AVERAGE: =AVERAGE(C2:C12)
  • MAX: =MAX(C2:C12)
  • MIN: =MIN(C2:C12)
  • ROUND: =ROUND(A2,1)
  • ROUNDUP: =ROUNDUP(A2,0)
  • ROUNDDOWN: =ROUNDDOWN(A2,0)
  • COUNT: =COUNT(C2:C12)
  • COUNTA: =COUNTA(C2:C12)
  • COUNTIF: =COUNTIF(C2:C12,">75")

C. Date & Time Functions

  • TODAY: =TODAY()
  • NOW: =NOW()
  • DATE: =DATE(2025,11,16)
  • DATEDIF: =DATEDIF(B2,TODAY(),"Y")
  • MONTH: =MONTH(A2)

D. Logical Functions

  • IF: =IF(C2>=75,"Pass","Fail")
  • AND: =AND(C2>70,D2="Yes")
  • OR: =OR(A2="HR",A2="Finance")
  • Nested IF: =IF(C2>=80,"A",IF(C2>=60,"B","C"))

E. LOOKUP Functions

  • VLOOKUP: =VLOOKUP(B5, B5:D14, 3, FALSE)
  • HLOOKUP: =HLOOKUP(3, K4:N7, 3, FALSE)

F. Statistical Functions

  • MEDIAN: =MEDIAN(C2:C12)
  • MODE: =MODE(C2:C12)
  • STDEV: =STDEV(C2:C12)
  • VAR: =VAR(C2:C12)
