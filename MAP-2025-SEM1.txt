a. Introduction to Data Analysis & Excel Overview

i. Data Analysis :

Data Analysis is the systematic and meticulous process of observing or inspecting data, cleansing or cleaning of data, transforming data, and modeling data to uncover and reveal useful information, draw conclusions, and support informed decision-making. 

It is a critical component of data science in helping businesses and researchers translate raw numbers into actionable insights which finally helps in improving business, increasing profits, enhancing effeciciencies and attaining business goals with business intelligence.

ii. Data Analysis Process :

The process of data analysis is typically iterative or repeatative and involves several important stages:

a. Define Objectives: Clearly state the problem you are trying to solve or the specific questions the data needs to answer. This sets the scope and direction for the entire analysis.

b. Data Collection: Gather the relevant data from various sources like databases, excel spreadsheets, data surveys, or create your own data.

c. Data Cleaning/Wrangling: This is often the most time-consuming step. It involves preparing the data by:

	Handling missing values (e.g., deleting records or imputing data).

	Removing duplicates.

	Correcting inconsistencies and errors (e.g., misspelled categories or incorrect formats).

	Standardizing and transforming data to a usable format.

d. Data Analysis: Apply analytical and statistical techniques to the cleaned data to find patterns, relationships, and trends. This step includes:

	Four Types of Data Analysis

	A. Descriptive Analysis: Summarizing what happened (e.g., calculating mean, median, mode, or charting frequency). Understanding of data.

	B. Diagnostic Analysis: Determining why it happened (e.g., using correlation or regression). Determine the data.

	C. Predictive Analysis: Forecasting what might happen next (e.g., using trend analysis). Forecasting or future prediction. 

	D. Prescriptive Analysis: Recommending a course of action. Taking action.

Description, Diagnosis, Prediction, Prescription

e. Interpretation and Visualization: Translate the analytical results into meaningful insights and Data visualization by drawing charts, graphs, and dashboards which are essential to make complex data easy to understand for non-technical audiences and management staff.

f. Communication: Present the findings and conclusions in a clear, convincing narrative often called data storytelling to guide decision-making. Generally done by preparing powerpoint show.

Data Objectives, Data Collection, Data Cleaning, Data Analysis, Interpretion and Visualization, Communication 

- Data Analysis Cycle

## f. Lookup Functions (MOST IMPORTANT)

Vlookup(), hlookup(), xlookup()

1. VLOOKUP()

Searches some data in excel, vertically or column-wise in a range of cells.

=VLOOKUP(B5, B5:D14, 3, FALSE)

B5: This is the lookup_value. It's the specific value you want to find, in this case, a cell containing an student rollno.

B5:D14: This is the table_array. It's the range of cells where the lookup will take place. The VLOOKUP will only search within the first column of this range.

3: This is the col_index_num. It specifies which column in the table_array contains the value you want to return. Here, it's the third column.

FALSE: This is the range_lookup argument. FALSE (or 0) tells the function to find an exact match for the lookup_value

2. HLOOKUP()

Searches some data in excel, horizontally or row-wise in a range of cells.


To use HLOOKUP, data must be arranged horizontally, like this:

Get Name of Roll No = 1

=HLOOKUP(colno, K4:N7, 2, FALSE)

Get Age of Roll No = 3

=HLOOKUP(3, K4:N7, 3, FALSE)
